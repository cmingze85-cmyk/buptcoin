# 🚀 BuptCoin 快速启动指南

## 🎯 5分钟快速上手

### 第一步：环境准备

```bash
# 1. 克隆或进入项目目录
cd BuptCoin

# 2. 安装依赖
pip install -r requirements.txt

# 3. 验证 Python 版本 (需要 3.7+)
python --version
```

### 第二步：启动系统

#### 方案 A: 启动增强版 (推荐 ✨)

```bash
python gui_enhanced.py
```

#### 方案 B: 启动原始版

```bash
python gui.py
```

### 第三步：完成初始化

1. 启动对话框会检查数据库
2. 选择 "启动系统" 继续
3. 系统自动创建创世区块
4. 准备就绪！

## 📝 基本操作 (3个步骤)

### ① 创建交易

```
菜单位置: [💸 交易与挖矿] 标签页

操作步骤:
1. 发送方地址: 选择 "genesis"
2. 接收方地址: 输入任意地址 (例: "alice")
3. 转账金额: 输入 10
4. 点击 "🚀 发送交易"

成功标志: 
✅ 状态栏显示 "交易已发送"
✅ "待处理交易" 数字增加
```

### ② 挖矿

```
菜单位置: [💸 交易与挖矿] 标签页

操作步骤:
1. 矿工地址: 选择任意地址 (例: "miner1")
2. 点击 "🚀 开始挖矿"
3. 等待挖矿完成 (~1 秒)

成功标志:
✅ 状态显示 "✅ 挖矿完成"
✅ 新区块添加到区块链
✅ 矿工获得 10 BPC 奖励
✅ 交易被确认
```

### ③ 查看余额

```
菜单位置: [💰 余额管理] 标签页

查看内容:
✓ 所有地址列表
✓ 每个地址的当前余额
✓ 地址状态标识
✓ 总计余额显示
```

```

```

2. 确认:
   ✓ 交易时间正确
   ✓ 发送/接收地址正确
   ✓ 金额正确

```
1. 打开 [💰 余额管理]
2. 计算总和应等于:
   初始余额 + 挖矿奖励 - 手续费
   1000 + 10 + 0.1 = 1010.1 BPC
```

## 🆘 故障排除

### 问题 1: "数据库连接失败"

✅ 解决:
```
系统自动切换到内存模式
数据暂时不会保存到数据库
功能正常,但关闭后数据丢失
```

### 问题 2: "挖矿很慢"

✅ 解决:
```
难度等级 2 (默认) 应该 <1 秒
如果超过 5 秒,检查:
  ✓ CPU 占用率
  ✓ 是否有后台进程
  ✓ 系统性能
```

### 问题 3: "交易被拒绝"

✅ 解决:
```
检查错误信息:
  ✓ "余额不足" → 增加转账金额
  ✓ "地址相同" → 改变接收方地址
  ✓ "输入错误" → 检查输入格式
```

### 问题 4: "界面显示乱码"

✅ 解决:
```
检查系统字体:
  ✓ 确保安装了 Microsoft YaHei
  ✓ 或其他中文字体
  ✓ 重启应用
```
